#Scratch Werk

##Prerequisites

* [node](git config --global push.default simple)
* [bower](http://bower.io/)
* [grunt](http://gruntjs.com/)

##Getting started

1. clone repo
2. npm install (picks up packages.json file)
3. bower install (picks up bower.json file)
4. grunt bowercopy to install and copy components
